# 🌤️ Weather Trading Agent

AI-powered Polymarket weather prediction market trader built with Python.

## Features
- 🌍 Live weather data for 5 global cities (London, New York, Tokyo, Sydney, Mumbai)
- 🤖 AI-powered trading analysis using LLM (OpenRouter)
- 📊 Kelly Criterion risk management
- 📄 Paper trading mode
- 🖥️ Professional Streamlit dashboard

## Tech Stack
- Python
- OpenWeatherMap API
- OpenRouter (Cohere LLM)
- Apify
- Streamlit
- Polymarket Gamma API
